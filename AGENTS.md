# AGENTS.md - Public Portfolio

Dieses Repository ist das oeffentliche, recruiter-freundliche Portfolio von Zoe Hanke.

## Zweck

- Rollenprofile, Case Studies, Skills und Timeline fuer oeffentliche Einsicht.
- Kurze, public-safe Zusammenfassungen aus `../portfolio-private/`.
- Keine Rohdatenbasis und kein vollstaendiger interner Projekt-Audit.

## Regeln

- Keine Kundennamen, Reseller-Namen, Hostnames, internen URLs, Tickets, Rohdatenpfade oder Secrets.
- Keine frueheren Namen verwenden; oeffentlich nur `Zoe Hanke`.
- Inhalte aus `portfolio-private/` immer kuerzen, anonymisieren und auf Zielrollen zuschneiden.
- Gute oeffentliche Begriffe: `Hetzner Dedicated/Cloud`, `Checkmk Distributed/MSP`, `Monitoring-Service for customers/resellers`, `Veeam service-provider operations`, `AI-assisted Engineering`.
- Technische Details nur so konkret, dass Kompetenz sichtbar wird, aber keine internen Betriebsablaeufe offengelegt werden.

## Stil

- Englisch, klar, professionell, recruiter-freundlich.
- Konkrete Outcomes statt langer interner Listen.
- Case Studies nach Problem/Work/Technologies/Outcome.
